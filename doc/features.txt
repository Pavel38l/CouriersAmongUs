Список планируемых фич приложения:
  То что уже реализовано в других магазинах:
    Выбор платформы (Intel, Amd)
    Подбор сборок в заданном диапазоне цен
    Комментарии к уже имеющимся сборкам
    ?Возможность создать закрытую сборку с доступом по ссылке
    Проверка совместимости по параметрам (список ниже)
    
  То что будет уникальным
    ?Возможность ставить оценки открытым сборкам
    Цены из разных магазинов
    Встроенная барахолка по городам
    Разные сетапы (Только пк, пк+монитор+клавиатура+мышь = готовый сетап)
    Для процессоров тянуть счет из базы geekbench чтобы сразу на сайте оценивать производительность (Api похоже нет, но сайт довольно легкий и фича классная ИМХО)
    Если зайдет предыдущий пункт аналогично для видеокарт
    
   
 
Магазины
  DNS предоставляет прайс лист в Excel формате
  Citilink придется парсить
  Яндекс маркет api платный с 2018 года и, видимо, нам не подходит
  
Параметры проверки совместимости
  CPU-MB сокет
  MB-CASE форм-фактор (ATX, mini-ITX)
  M2 SSD - MB если разъема нет
  RAM DDR3-DDR4 + формат DIMM - SODIMM
  ?CPU-MB Предупреждение о фозможной несовместимости версий BIOS, если процессор поддерживается начиная с какой-то версии
  ?Длина видеокарты и параметры корпуса
  ?Разъемы питания видеокарты и разъемы на БП
  ?Предупреждение про рекомендованную мощность БП для видеокарты
  ?Предупреждение про питание процессора, если от БП 4pin а на плате 8pin
