Магазины
  DNS предоставляет прайс лист в Excel формате
  Citilink придется парсить
  Яндекс маркет api платный с 2018 года и, видимо, нам не подходит
  
Параметры проверки совместимости
  CPU-MB сокет
  MB-CASE форм-фактор (ATX, mini-ITX)
  M2 SSD - MB если разъема нет
  RAM DDR3-DDR4 + формат DIMM - SODIMM
  ?Максимальная частота RAM в MB
  ?CPU-MB Предупреждение о фозможной несовместимости версий BIOS, если процессор поддерживается начиная с какой-то версии
  Параметры корпуса(длинны видеокарт, высота кулера, размер БП, поддерживаемы форм факторы)
  ?Разъемы питания видеокарты и разъемы на БП
  ?Предупреждение про рекомендованную мощность БП для видеокарты
  ?Лининия питания в MB для процессора
  ?Предупреждение про питание процессора, если от БП 4pin а на плате 8pin
